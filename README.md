## Información
Pokephone es un proyecto con la intención de condensar en él toda la información sobre Pokemons, objetos, mapas, y mas información sobre los videojuegos con una IU facil de usar para cualquier persona y que puedan utilizarlo como una guía de uso rápido y cómodo.

🚧 --> Proyecto aún en construcción <-- 🚧


## 🔨 Funcionalidades del proyecto
- `Funcionalidad 1`: Lista infinite scroll con tarjetas de los 1025 pokemons actuales.
- `Funcionalidad 2`: Hoja de información con los datos de cada pokemon visible al hacer click sobre su tarjeta.
- `Funcionalidad 3`: Busqueda de pokemon por nombre.
- `Funcionalidad 4`: Sistema de filtrado de pokemons según tipo y generación.

🚧 Proximas funciones 🚧

- `Funcionalidad 5`: Marcar pokemons favoritos y poder verlos en una lista separada a los demás.
- `Funcionalidad 6`: Menú con los diferente mapas de cada región (Kanto, Johto, Hoenn...) que al hacer click sobre las diferentes zonas saldrá información sobre esa ruta, ciudad o zona en cuestión.
- `Funcionalidad 7`: Lista de objetos que aparecen en los juegos para poder buscar y filtrar los que quieras y leer información sobre ellos y como conseguirlos.

## Instrucciones de uso
### 📁 Acceso al proyecto
- En primer lugar deberá tener instalado las funciones de git en el ordenador donde see vaya a utilizar el repositorio
- Deberá clonar el repositorio en su ordenador escribiendo el la consola de comandos del ordenador en la ruta que desee hacerlo "git clone https://github.com/zaxwtf/PokePhone.git
### 💻 Abre y ejecuta el proyecto
- Deberá entrar en la carpeta donde se haya clonado el repositorio y sobre el archivo index.html hacer click derecho/ abrir con/ y eliga el navegador que desee usar
### 🛠️ Configurar archivos
- Usando un editor de código como por ejemplo VisualStudio podrá ver todos los archivos del repositorio, en index.html encontrará el esquema básico html de la web y poco más, la mayoria de la página se construye desde JavaScript
- En la carpeta css encontrará el archivo style.css donde verá todos los estilos para cada sección de la aplicación
- script.js es el documento JavaScript donde se encuentra absolutamente todo el funcionamiento de la aplicación y gran parte del HTML que se construye de ahí **⚠️CUIDADO CON ESTE ARCHIVO⚠️**
